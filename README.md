This project focuses on analyzing student academic performance across multiple semesters using Python.

Dataset
The dataset contains student marks for multiple semesters with different subjects.

**File included:**
- `student_exam_performance_sem1_sem2_sem3.csv`

- **Semester 2:** Maths 2, Civil, C++  
- **Semester 3:** DSA, Python, DELD  



 Tools & Technologies Used
- Python  
- Pandas  
- Matplotlib  
- NumPy  
- Google Colab  



 Features of the Project
- Data loading and cleaning
- Semester-wise performance comparison
- Subject-wise analysis
- Graphical visualization using Matplotlib
- Easy-to-understand insights from data



 How to Run the Project
1. Open the project in **Google Colab**
2. Upload the CSV dataset if required
3. Run the notebook cells step by step
4. View the generated analysis and graphs

- Comparative analysis across semesters
- Clear insights into strengths and weak subjects
